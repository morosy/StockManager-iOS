# AGENTS.md

## コーディング規約
- インデントはスペース4つ
- if文などは1行でも必ず波括弧を付ける
- 既存の命名規則を維持する
- 不要なリファクタはしない
- 文字コードはUTF-8
- 改行文字はCRLFを用いる
- 実装した内容を `.prompt/document.md` へ記述する

## 作業手順
- 変更前に関連ファイルを確認する
- 変更後は lint, test, build を実行する
- 失敗時は原因を説明し、最小修正で再試行する

## 禁止事項
- 新規依存の追加は明示指示があるときのみ
- 大規模なファイル移動は禁止
- 公開APIの変更は禁止
